#星云(Nebula)#
星云(英文名：Nebula)是一个MC的mod，提供了一些库供其它mod使用。
Nebula is a modification of Minecraft which provide some library to other mods. 
目前Nebula由作者ueyudiud开发
The author is ueyudiud who is creating the mod.
